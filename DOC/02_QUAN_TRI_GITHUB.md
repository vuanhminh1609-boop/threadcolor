## Project board đề xuất
Tạo GitHub Project với các cột:
- Backlog
- Đang làm
- Review
- Sẵn sàng Deploy
- Done

## Quy ước quản trị
- Mỗi PR phải gắn ít nhất 1 label P0/P1/P2.
- Mỗi issue lớn nên có checklist test nhanh trong mô tả.
